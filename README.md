# yolog.dev support

Welcome to the official support repository for [yolog](https://yolog.dev).

## Downloads

Download the latest Yolog Desktop app from the [Releases](https://github.com/yologdev/support/releases) page.

### Platforms
- **macOS** (Apple Silicon & Intel) - `.dmg`
- **Windows** (x64) - `.exe`
- **Linux** (x64) - `.AppImage`

## Feedback & Support

Have questions, feature requests, or found a bug? [Open an issue](https://github.com/yologdev/support/issues/new).
